# Edge Case Recipes

This directory contains hand-curated recipes that cover edge cases
for testing the recommendation engine:

- Recipes with unusual dietary combinations (e.g. vegan + high-protein)
- Multi-cuisine fusion dishes
- Recipes with many optional ingredient substitutions
- Very short (≤5 min) and very long (≥3 hour) preparation times
- Recipes with rare or regional ingredients

Files are named `{id}_{slug}.json` and conform to the Recipe model schema.
